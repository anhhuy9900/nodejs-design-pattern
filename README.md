
# Nodejs Design Pattern

A project for practice with design pattern



## Structure

This project is used by the following patterns:

- Creational
    - Factory
        - src/factory-method
    - Abstract Factory
        - src/abstract-factory
    - Builder
        - src/builder
    - ProtoType
    - Singleton
        - src/singleton
- Structural
    - Decorator
    - Adapter
        - src/adapter
    - Facade
        - src/facade
    - Bridge
    - Composite
    - Flyweight
    - Proxy
- Behavioral
    - Command
    - Chain of Responsibility
    - Observer
    - Interpreter
    - Iterator
    - Mediator
    - Memento
    - State
    - Strategy
    - Template Method
    - Visitor

- Solid
    - Dependency Injection
        - src/dependency-injection
    - SOLID Principle
        - src/solid-design-principle
